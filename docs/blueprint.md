# **App Name**: NEU LibLog

## Core Features:

- Secure User Authentication: Allow students and faculty to log in using their institutional email (Google-based) or by scanning a QR code ID for a 'Tap ID' experience.
- Purpose of Visit & College Input: Provide a simple interface for users to quickly input their reason for visiting the library and their college affiliation upon entry.
- Dynamic Admin Dashboard: Present administrators with a real-time overview of library usage statistics, including daily, weekly, and monthly entries, filterable by student name or ID number.
- Student Visit Log History: Enable administrators to view detailed historical logs of student and faculty visits, allowing search by name or ID, and filtering by day, week, month, including specific log-in times.
- Admin User Management: Empower administrators to manage user access, specifically the ability to block or unblock users from the system.
- Role-Based Access Control: Implement distinct access levels and functionalities for 'Administrator' and 'User' (student/faculty) roles after successful login.
- AI Reason Categorization Tool: Provide an AI tool to help users select more specific visit reasons or assist administrators in categorizing diverse free-text reasons into consistent groups for better statistical reporting.

## Style Guidelines:

- Primary color: #229CCD. A professional and calming blue, symbolizing clarity and knowledge, setting an academic tone for key interactive elements.
- Background color: #EAF6F9. A very light, desaturated blue-gray to provide a clean and bright backdrop that ensures readability and a focused environment.
- Accent color: #33DDAA. A vibrant aqua-turquoise that is analogous to the primary, used for highlighting important actions, notifications, and interactive elements to draw user attention.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly readable, and neutral characteristics, suitable for an institutional application.
- Utilize a consistent set of simple, line-art style icons for clarity and professionalism, aiding quick recognition in the dashboard and user interface.
- Implement a clean and organized layout with a central login dialog, and an administrative dashboard featuring distinct card-based sections for usage statistics, ensuring efficient information processing.
- Incorporate subtle and smooth transition animations for navigation and loading states, along with gentle confirmations for successful actions like login, enhancing user feedback without distraction.
